# AI Agent 目录约束

本目录用于组织 SilentLake 的 AI Agent 资产、扩展能力和代理协作说明。

约束：

- 本目录遵循根目录 `AGENTS.md` 的全部规则。
- 本目录中的软链接仅做结构映射，不修改真实 agent 运行入口。
- 任何 agent、插件、子代理、运行协议或权限能力的变更，都应补充前端表现、用户感知和后端工作原理说明。
- `AGENTS.md` 与 `CLAUDE.md` 必须保持完全一致。
