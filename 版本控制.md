# 版本控制

## 当前对外版本

- `package.json` 当前版本：`3.0.0`

## 当前版本说明

### 3.0.0

变更类型：

- 大版本架构升级
- 控制台前端全量迁移
- 旧 OpenClaw 核心控制功能 React 化
- SilentLake 新旧功能统一 UI 重设计

版本判定：

- `1.0.0` 为首个稳定归档版本
- `2.0.0` 为 React 主架构切换版本
- `2.1.0` 为任务链、MCP 与工程链路收口版本
- 本轮将旧 OpenClaw 保留控制台能力整体迁入 React 主线，并完成统一设计系统重做，属于架构级与产品级重构，因此大版本升为 `3.0.0`

影响范围：

- `ui` React 控制台主工程、导航结构、数据层和所有主页面
- Gateway 控制能力在浏览器端的接入方式与状态管理
- README、版本展示、版本归档目录
